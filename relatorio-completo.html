1. Servidor DHCP (ISC-DHCP-Server)

Configuração inicial para automatizar a entrega de IPs na rede e definir o servidor DNS padrão.

Configuração do Escopo (dhcpd.conf)

Definição da subnet 10.40.40.0/24, gateway e apontamento para o DNS local.

subnet 10.40.40.0 netmask 255.255.255.0 {
  range 10.40.40.100 10.40.40.200;
  option routers 10.40.40.1;
  option domain-name-servers 10.40.40.1;
  option domain-name "grenal.edu";
}

Reserva de IP (Segurança)

Fixação de IP para um host específico via endereço MAC.

host juvenil {
  hardware ethernet 00:08:54:25:73:1a;
  fixed-address 10.40.40.157;
}
