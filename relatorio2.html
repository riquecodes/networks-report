Servidor Linux: DNS, DHCP e Web Server

1. Instalação e Configuração do DHCP

Abaixo descrevemos o processo técnico realizado para configurar o servidor de endereçamento automático.

1.1 Instalação do Pacote

sudo apt-get install isc-dhcp-server

1.2 Definição da Interface

Edição do arquivo /etc/default/isc-dhcp-server para definir qual placa de rede distribuirá os IPs.

sudo nano /etc/default/isc-dhcp-server

1.3 Configuração do Arquivo Principal (dhcpd.conf)

Configuração do arquivo /etc/dhcp/dhcpd.conf com as regras da sub-rede, gateway, DNS e segurança.

# Configurações globais de segurança e gerenciamento ignore-client-uids true; deny duplicates; one-lease-per-client on; # Configuração da Sub-rede (Faixa de IPs) subnet 10.40.40.0 netmask 255.255.255.0 { range 10.40.40.100 10.40.40.200; # Intervalo de IPs para clientes option routers 10.40.40.1; # Gateway Padrão option domain-name-servers 10.40.40.1; # Servidor DNS (Próprio Linux) option domain-name "grenal.edu"; # Nome do Domínio } # Reserva de IP (Host Específico - Exemplo) host juvenil { hardware ethernet 00:08:54:25:73:1a; filename "msh.juvenil"; server-name "juvenil.tirubico.com"; fixed-address 10.40.40.157; }

1.4 Aplicação das Mudanças

Após salvar as configurações (Ctrl+X, Yes), o serviço foi reiniciado e verificado.

sudo service isc-dhcp-server restart
sudo service isc-dhcp-server status
